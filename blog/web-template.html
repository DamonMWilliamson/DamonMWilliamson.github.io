Hosted Website Template

I, after working on my website decided that I should make an open source template for anyone to use to make a website and learn html, css, js, and other server stuffs

How to use this Template

  1. fork the template and open it in github desktop
  2. find the index.html
  3. replace all the place holders
  4. repete for other top level pages
  5. open the style.css
  6. change the background image and theme colours
  7. copy the example article
  8. modify it
  9. update the RSS.xml to whare your website is hosted
  10. change the links in the contact me
  11. add some images to the photo site
  12. update photo-RSS.xml
  13. open friends.csv
  14. add your friends websites
  15. save evreything
  16. upload it to your host
    for more info on hosting (getting your website on the internet) view this article I made
  17. share with friends

If you have any trouble please contact me and I will try to help you to the best of my ability
